# Installation
* Requirements : **Visual Studio Code**
- Use the terminal to download the source code:: git clone https://github.com/letronghieubkhn-droid/flappy-bird.git
- In the terminal, use the following command to build the project: make
- After that, the game will start automatically..
- If you encounter any issues during installation or while playing, please contact: letronghieubkhn@gmail.com.
